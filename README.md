This repository contains synthetic tweet datasets generated using an agentic workflow designed to produce crisis-related social media data. 

The datasets correspond to six real-world earthquake events:
1. 2014 Napa, California earthquake, 
2. 2014 Iquique, Chile earthquake,
3. 2015 Nepal earthquake,
4. 2019 Ridgecrest, California earthquake,
5. 2021 Fukushima, Japan earthquake, and
6. 2021 Haiti earthquake. 

The synthetic tweets were generated to reflect prespecified target labels for location and damage level, enabling experimentation with AI systems for geolocalization and damage level prediction tasks. These synthetic tweet datasets are intended to support research in crisis informatics, particularly in scenarios where real-world tweet datasets are unlabeled, limited, or unavailable.
